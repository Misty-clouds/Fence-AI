# Map Page User Guide

## Overview
The Map Page provides an interactive map interface for discovering and exploring locations. It integrates Google Maps with powerful search and drawing capabilities.

## Features

### 1. Search for Locations
- **Search Bar**: Located at the top of the screen
- Type any location, address, or place name
- Get instant search results with addresses
- Tap a result to navigate to that location on the map

### 2. Draw Mode
- **Enable Drawing**: Tap the "Draw to search" button at the bottom
- **Draw a Polygon**: Tap on the map to create points
- **Complete Drawing**: After placing 3+ points, tap the search icon
- **Cancel Drawing**: Tap the X button to cancel
- The app will search for information about the area you've drawn

### 3. Map Controls

#### Zoom Controls (Left Side)
- **+ Button**: Zoom in
- **- Button**: Zoom out
- **Location Button**: Go to your current location

#### Top Bar
- **Menu Icon**: Open navigation drawer
- **Chat Icon**: Access messages/chat

### 4. Location Information
When you search for an area (either by text or drawing), you'll see:
- **Address Details**: Full formatted address
- **Coordinates**: Latitude and longitude
- **Elevation**: Height above sea level
- **Nearby Businesses**: Restaurants, shops, services within 2km
- **Nearby Schools**: Educational institutions within 3km
- **Nearby Hospitals**: Medical facilities within 5km

### 5. Visual Elements
- **Green Markers**: Your drawn polygon points
- **Red Marker**: Selected location
- **Shaded Area**: Your drawn search polygon
- **Blue Dot**: Your current location (when available)

## How to Use

### Basic Search Flow
1. Open the Map Page
2. Type a location in the search bar
3. Select from the results
4. View location details in the bottom sheet

### Draw and Search Flow
1. Tap "Draw to search" button
2. Tap multiple points on the map to create a polygon
3. Tap the search icon when done (minimum 3 points)
4. View comprehensive data about the selected area

### Navigation Flow
1. Use zoom controls to adjust view
2. Tap "My Location" to center on your position
3. Drag the map to explore different areas
4. Pinch to zoom (mobile devices)

## Tips
- Grant location permissions for the best experience
- Draw larger areas for broader property searches
- Use the search bar for specific addresses
- Check nearby amenities to understand the neighborhood

## Permissions Required
- **Location Access**: To show your current position
- **Internet**: To load map tiles and search data

## Troubleshooting
- **Map not loading**: Check internet connection
- **Location not showing**: Enable location services in device settings
- **Search not working**: Verify internet connection and API key configuration
